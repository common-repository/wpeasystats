== Description ==

**Get all your web statistic ,PR, back links, indexing and google keyword position at your fingertips.**

* site PR 
* How many pages are indexed (Google, Yahoo, Bing)
* How Many Back links are pointing your site
* Visitors, Now you can know exactly who are your visitors: Date and Time, IP, Country, Language, OS Browsers
* Spiders, You now can find out which spiders visits your site and how often
* Search keywords
* Referrers

you also track how your keywords ranked in Google. for each keyword you decide put in, this module will check if it ranked at top 100 results, it will show you the exact place you are ranked

 
== Installation ==

This section describes how to install the plugin and get it working.

*   Download the plugin
*   Extract the downloaded zip file to your local hard drive
*   Upload the "wpeasystats" plugins directory to your blog (to the wp-content/plugins directory)
*   Activate the plugin
